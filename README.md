This project focuses on performing Exploratory Data Analysis (EDA) on a police dataset using Python and Pandas in Jupyter Notebook.

The dataset contains information related to traffic stops, driver demographics, violations, searches conducted, stop duration, and other police-related records.

The main objective of this project is to analyze real-world police data, uncover useful insights, and practice data analysis techniques using Pandas.

📂 𝗗𝗮𝘁𝗮𝘀𝗲𝘁 𝗜𝗻𝗳𝗼𝗿𝗺𝗮𝘁𝗶𝗼𝗻:

The dataset used in this project contains information about police traffic stops and driver-related details. It includes various columns such as driver gender, driver age, violation type, search conducted, stop duration, drug-related stops, and traffic violations.

The dataset provides a real-world scenario for analyzing traffic stop records and identifying behavioral patterns. By working with this dataset, different analytical operations can be performed to understand which types of violations occur more frequently, how searches are conducted, and how driver demographics affect traffic stop outcomes.

📊 𝗞𝗲𝘆 𝗢𝗽𝗲𝗿𝗮𝘁𝗶𝗼𝗻𝘀 𝗣𝗲𝗿𝗳𝗼𝗿𝗺𝗲𝗱

Several important data analysis operations were performed during this project. The first step involved data cleaning, where missing values were identified and handled properly. Unnecessary columns were removed, and column names were formatted for better readability and analysis.

After cleaning the dataset, various exploratory data analysis techniques were applied. Filtering and sorting operations were used to extract meaningful subsets of data. GroupBy functions were applied to compare categories such as gender, violation types, and search rates.

Different visualizations were created to better represent the data and identify hidden patterns. The project also included checking data types, statistical summaries, unique values, and relationships between columns. These operations helped in gaining a deeper understanding of the dataset.


🔍 𝗣𝗿𝗼𝗷𝗲𝗰𝘁 𝗜𝗻𝘀𝗶𝗴𝗵𝘁𝘀

The analysis performed in this project helped uncover several useful insights from the police dataset. It identified which gender was stopped more frequently and which types of traffic violations were most common among drivers.
The project also analyzed how often vehicle searches were conducted during traffic stops and compared search rates across different categories. Stop duration analysis helped understand the frequency of short and long stops. Drug-related stop records were also examined to identify patterns and trends.

By comparing different driver demographics and traffic violations, the project demonstrated how data analysis can help discover meaningful information from large datasets.
